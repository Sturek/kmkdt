# KMKDT
Die komische MKD-Tastatur, ein Community-Projekt der #komische-tastaturen Kanäle im MKD-Discord ( https://discord.gg/AnKuj8Y ). Der entsprechende Thread ist "Komische Channel Board".

## Allgemein
Die Features dieser Tastatur wurden durch vier Umfragen festgelegt. Die grundlegeden Eigenschaften stehen also fest, das genaue Design wird in einem informellen Prozess gemeinschaftlich erstellt. Dazu dient dieses Repo.

## Eigenschaften aus Umfragen
- 3 Row, 12u Spalten
- Unibody Split mit Columnar Stagger
- 1u Mods
- Pinky + Ring Splay
- 3 Thumbs pro Seite als Fächer
- MX switches
- Onboard MCU
- Daughterboard
- Trackpad/Trackball
- Flip Switch
- 2 Encoder
- 1 Display
- Magnetische Pogopins
- Status-LEDs
- Tenting
- Badges
- Case Varianten (noch offen)

## Das Repo
Wer Collaborator werden möchte kann sich bei @DerBonk melden. Die meisten Dateiformate, die wir brauchen, werden nicht gut mit git funktionieren. Bitte nutzt daher die Checkout-Datei und tragt euch ein und aus, damit es keine Dopplungen oder Konflikte zwischen den Dateien gibt.

In "Aktueller Stand" findet ihr eine Zusammenfassung der derzeitigen Ideen und eine Liste was noch zu tun ist. Freiweillige können sich dort auch schon im Vorhinein für einzelne Aufgaben melden.

Im Moment arbeiten wir nur im "main" branch, um die Dinge einfach zu halten.
